### Comandos para Markdown

- `#` Se utiliza para títulos o encabezados. Entre más `#`, más pequeño el título.
- `**texto**` Se utiliza para el texto en **negrita**.
- `*texto*` Se utiliza para el texto en *cursiva*.
- `~~texto~~` Es para ~~tachar~~ texto.
- `` `código` `` Se usa para poner código en línea dentro de una frase.
- ```` ``` ```` Se usa para poner bloques de código de varias líneas.
- `>` Se usa para una cita o bloque de texto.
- `-` Genera una lista con viñetas.
- `1.` Genera una lista enumerada.
- `[texto](url)` Inserta un enlace.
- `![alt](url)` Inserta una imagen.
- `| ... | ... |` Se usa para crear tablas con filas y columnas.
- `---` Inserta una línea horizontal como separador.
